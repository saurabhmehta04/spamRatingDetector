Operating System & Tools required:

1. Ubuntu 12.04 and above
2. Hadoop 2.2.0 and above
3. Mahout 0.9
4. Eclipse
5. Maven 3.0 and above

Project Folders Description:

1. MyMahout-Project : Contains the java source files (Parsing data, Running the server, 		      Classifier & CSV to Sequence file converter).
2. spamseqfile 	    : Sequence file of the dataset required by mahout.
3. PROJECT	    : Files stored on the hadoop file system.
4. SPAMDATASET.csv  : Training dataset.
5. User-Interface   : Folder containing the JavaScript, HTML5, CSS3 & PHP files for the user interface.
